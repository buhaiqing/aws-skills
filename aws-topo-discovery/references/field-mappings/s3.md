# S3 Bucket Field Mapping

**AWS API**: `s3api list-buckets` -> `aws_s3_bucket`

## Mapping Rules

| HCL Attribute | API JSON Path | Type | Required | Notes |
|---------------|---------------|------|----------|-------|
| `bucket` | `Name` | string | Y | Block name derived from this |
| `tags` | (from `get-bucket-tagging`) | map | N | Name tag |

## Block Name

`{bucket_name_slug}` (e.g. `prod_data_bucket`)

## Stable Import ID

`{bucket_name}` (e.g. `prod-data-bucket`)

## Deferred to Phase 2

- Versioning (`aws_s3_bucket_versioning`)
- Encryption (`aws_s3_bucket_server_side_encryption_configuration`)
- Lifecycle rules (`aws_s3_bucket_lifecycle_configuration`)
- Public access block (`aws_s3_bucket_public_access_block`)
- Bucket policy (`aws_s3_bucket_policy`)

## Notes

- `list-buckets` returns all buckets globally (not region-scoped)
- Region requires separate `get-bucket-location` call per bucket
- Bucket name is globally unique
